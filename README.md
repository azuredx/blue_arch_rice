## Компоненты
- **OS:** EndeavourOS (Arch-based)
- **WM:** [Hyprland](https://hyprland.org/) (Wayland)
- **Bar:** Waybar
- **Terminal:** Kitty
- **Shell:** Zsh 
- **Runner:** Rofi
