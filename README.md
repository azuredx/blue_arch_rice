# 🦈 blueshark_arch

Мой персональный "rice" на базе **EndeavourOS** (Arch Linux). Сборка ориентирована на эстетику, минимализм и производительность под Wayland.

> `![Desktop Preview](screenshots/preview.png)`

## 🛠 Компоненты
- **OS:** EndeavourOS (Arch-based)
- **WM:** [Hyprland](https://hyprland.org/) (Wayland)
- **Bar:** Waybar
- **Terminal:** Kitty
- **Shell:** Zsh 
- **Runner:** Rofi
